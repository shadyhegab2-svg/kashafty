# كشافتي — تعليمات التشغيل (المرحلة ١)

## ١. إنشاء مشروع Firebase جديد
مهم: لازم يكون **مشروع منفصل تمامًا** عن مشروع أسد البحار (asad-elbahr).

1. روح https://console.firebase.google.com
2. "إضافة مشروع" → اسمه مثلاً `kashafty`
3. من القائمة الجانبية: **Firestore Database** → "إنشاء قاعدة بيانات" → اختر "وضع الإنتاج" (Production mode)
4. من إعدادات المشروع (⚙️ → Project settings) → "إضافة تطبيق ويب" (</>) → هياخدك لكود فيه `firebaseConfig` object

## ٢. لصق إعدادات Firebase في الكود
افتح `index.html`، دور على السطر ده تقريبًا:
```js
const fb = initializeApp({
  apiKey: "PASTE_YOUR_API_KEY",
  ...
});
```
واستبدل كل قيمة بالقيمة الحقيقية اللي جالك من Firebase.

## ٣. رفع قواعد الحماية (Firestore Rules)
1. من Firebase Console → Firestore Database → تبويب "Rules"
2. انسخ محتوى ملف `firestore.rules` والصقه هناك
3. "نشر" (Publish)

⚠️ **ملاحظة مهمة**: القواعد الحالية فيها فتحة مؤقتة (القراءة/الكتابة مفتوحة لأي حد) لحد ما نضيف Firebase Authentication حقيقي بدل نظام اسم المستخدم/كلمة المرور اليدوي. **متطلقش المنصة لمجموعات حقيقية غريبة عنك قبل ما نقفل النقطة دي**، عشان أي حد يقدر يشوف بيانات أي مجموعة تانية دلوقتي.

## ٤. رفع الملفات على GitHub
1. اعمل ريبو جديد: `shadyhegab2-svg/kashafty`
2. ارفع الملفات دي كلها في نفس المجلد الرئيسي:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon.svg`, `icon-512.png`, `icon-192.png`, `apple-touch-icon.png`, `favicon.ico`
3. من إعدادات الريبو (Settings) → Pages → فعّل GitHub Pages من فرع `main`

## ٥. تجربة النظام
- افتح الموقع → "سجّل مجموعتك الكشفية الآن"
- املأ البيانات واختار نوع ولون المجموعة
- هيديك "كود المجموعة" — ده اللي أي عضو إدارة في المجموعة دي هيستخدمه للدخول
- جرّب تسجيل الدخول بنفس الرقم/الباسورد اللي سجلت بيهم

## الخطوات الجاية
المرحلة دي بتغطي بس: التسجيل + تسجيل الدخول + إعدادات المجموعة (الاسم واللون).
باقي الوحدات (الأعضاء، المهام، الشات، الإعلانات...) هتتبني فوق نفس هيكل
`groups/{groupId}/...` في المراحل الجاية، ونقفل قواعد الحماية بشكل نهائي قبل أي إطلاق حقيقي.
